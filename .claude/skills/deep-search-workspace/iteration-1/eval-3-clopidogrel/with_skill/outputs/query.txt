=== HIERARQUIA DE FONTES (seguir esta ordem de prioridade) ===

1. GUIDELINES: Guidelines vigentes de sociedades médicas (EASL, AASLD, AGA, ACG, ESC, AHA, WHO, Cochrane, SBH, SBC).
   Extrair: sociedade, ano, grau de recomendação, nível de evidência, recomendação específica.

2. META-ANÁLISES: Revisões sistemáticas com meta-análise (preferencialmente >=3 estudos pooled).
   Extrair: k estudos, n pacientes total, estimativa pooled + IC 95%, I-squared, certeza GRADE.

3. RCTs LANDMARK: Ensaios clínicos randomizados multicêntricos (preferencialmente n>=200).
   Extrair: acrônimo, n, braços, endpoint primário, effect size + IC 95%, NNT se calculável, p-value.

4. LIVROS-REFERÊNCIA: Edições mais recentes apenas.
   Extrair: autor, edição, capítulo, páginas, argumento central do trecho.

5. COORTES GRANDES: Apenas quando não há evidência de níveis superiores.
   Extrair: n, follow-up, desfecho, medida de associação + IC 95%.

=== OLHAR CRÍTICO — aplicar a CADA achado ===

Para cada evidência encontrada, avaliar criticamente:
- População do estudo vs contexto do ensino (generalizável? spectrum bias?)
- Endpoint: clínico (mortalidade, evento) ou surrogate (marcador laboratorial)?
- Financiamento: indústria farmacêutica? Potencial viés?
- Replicação: achado confirmado por estudos independentes?
- Temporalidade: dado ainda vigente ou superseded por evidência mais recente?

Quando um achado é robusto, dizer que é robusto. Quando é frágil, dizer por quê.
O objetivo não é demolir — é ser preciso sobre o grau de confiança.

=== DIVERGÊNCIAS E CONVERGÊNCIAS (obrigatório quando 2+ fontes tocam o tema) ===

CONVERGÊNCIAS: Onde 2+ sociedades/estudos concordam → alta confiança. Listar com fontes.
DIVERGÊNCIAS: Onde discordam → listar posição de cada, ano, base de evidência, qual é mais recente.
Divergências mudam conduta clínica — omiti-las é desinformar.

=== REQUISITOS DE DADOS ===

Cada achado reportado precisa incluir:
- Números concretos (effect size, IC 95%, p-value, n, NNT quando aplicável)
- Identificação da fonte (PMID, DOI, ou citação completa)
- Ano de publicação
- Tag de tipo: [GUIDELINE] [META/SR] [RCT] [LANDMARK] [COHORT] [BOOK] [EXPERT]
- Qualidade GRADE: HIGH / MODERATE / LOW / VERY LOW

Achados sem números concretos não são aceitáveis.
"Estudos mostram que..." sem citação = excluir do output.

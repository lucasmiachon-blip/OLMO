## MELD Timeline

- 2000: Malinchoc -- TIPS model (bilirubin, creatinine, INR, etiology)
- 2001: Kamath -- validated for ESLD (dropped etiology)
- 2002 Feb 27: UNOS adopted MELD for liver allocation
- 2003: Wiesner -- prospective OPTN validation, mortality bands
- 2006: Biggins -- MELD-Na formula
- 2006 Jul (Portaria GM 1160, 29 May): Brazil adopted MELD
- 2008: Kim -- MELD-Na NEJM validation
- 2016 Jan 11: UNOS adopted MELD-Na
- 2021: Kim -- MELD 3.0 proposed
- 2023 Jul 13: OPTN implemented MELD 3.0
- 2024: GEMA-Na validated in Spain, Australia, Italy, UK (not adopted)
- 2025 Mar: Germany adopted reMELD-Na (NOT MELD 3.0)
- 2026 Mar: Brazil still uses original MELD + MELD-Na as special criteria

## MELD 3.0 Formula Details (verified 2026-03-31, 4 independent sources)

### Complete Formula (Kim WR et al. Gastroenterology 2021, PMID 34481845)

```
MELD 3.0 = round(
  1.33  * (if Female)
+ 4.56  * ln(Bilirubin)
+ 0.82  * (137 - Na)
- 0.24  * (137 - Na) * ln(Bilirubin)      [interaction]
+ 9.09  * ln(INR)
+ 11.14 * ln(Creatinine)
+ 1.85  * (3.5 - Albumin)
- 1.83  * (3.5 - Albumin) * ln(Creatinine) [interaction]
+ 6                                         [constant/floor]
)
```

### Variable Clamping

| Variable | Floor | Ceiling | Notes |
|----------|-------|---------|-------|
| Bilirubin (mg/dL) | 1.0 | none | ln >= 0 |
| INR | 1.0 | none | ln >= 0 |
| Creatinine (mg/dL) | 1.0 | 3.0 | Cap lowered from 4.0 in MELD-Na |
| Sodium (mEq/L) | 125 | 137 | |
| Albumin (g/dL) | 1.5 | 3.5 | |
| Sex | 0 (male) | 1 (female) | +1.33 for women |

Dialysis rule (OPTN): Cr automatically set to 3.0 mg/dL if 2+ dialysis in prior week or 24h CVVHD.
Score range: 6 (mathematical floor from +6 constant) to 40 (OPTN policy cap; raw can exceed 40).

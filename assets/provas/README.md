# Provas Reais — R3 Clinica Medica

PDFs de provas de bancas profissionais para analise de padroes e geracao de questoes calibradas.

## Uso

1. Colocar PDFs aqui
2. Claude analisa padroes (formato, temas, dificuldade, estilo da banca)
3. Topicos errados → gera questoes no mesmo formato (anti-cue protocol)
4. Output → Anki cards via MCP

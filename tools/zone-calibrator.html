Zone Calibrator

Como usar:
1. Selecione o forest plot acima
2. Arraste as zones para alinhar com as colunas
3. Use os handles (bordas) para redimensionar
4. Clique "Copy CSS" quando pronto

Click = selecionar zone
Drag = mover
Handles = redimensionar
Arrow = mover 0.5% (hold Shift = 0.1%)
Ctrl+Arrow = resize 0.5% (hold Shift = 0.1%)
V = toggle visibilidade da zone selecionada
